C/C++

C/C++ 开发示例

Docklet提供了GCC 5+ 和 GDB，用于帮助开发 C 程序。

C

文件 sum.c 如下:

#include <stdio.h>
#define TOP 50

int main()
{
    int i = 0 , sum = 0 ;
    while ( i <= TOP ) {
        sum += i ;
        i++ ;
    }
    printf(" Sum of 1 to %d is %d\n", TOP, sum ) ;
    return 0 ;
}

进入WEB Terminal，编译并运行 

$ gcc -g sum.c -o sum
$ ./sum
 Sum of 1 to 50 is 1275

C++

下面显示了一个有错的C++程序main.cpp，原文来源于这里

#include<iostream>

using namespace std;

long factorial(int n);

int main()
{
    int n(0);
    cin>>n;
    long val=factorial(n);
    cout<<val;
    cin.get();
    return 0;
}

long factorial(int n)
{
    long result(1);
    while(n--)
    {
        result*=n;
    }
    return result;
}

进入 WEB Terminal，编译并运行，输入 5 ，结果 0

$ g++ -g main.cpp -o main
$ ./main

0

此时 result变成了 2。我们已经发现了一个bug: result最终的值应为 3*2*1， 但这里只是从 2 开始计算，问题出在 while (n--)上。

将代码修改为如下

while(n>0) //doesn't let n reach 0
{
  result*=n;
  n--;        //decrements only after the evaluation
}

测试运行通过。
